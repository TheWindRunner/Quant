# 免费数据源方案

## 完整基金净值

首选：AKShare `fund_open_fund_info_em`

- 指标选择“单位净值走势”时返回基金成立以来完整日净值。
- 免费，无月费。
- 本项目已将结果缓存到`data/nav_cache`，避免重复请求和接口波动。

备用：东方财富公开基金页`pingzhongdata/{基金代码}.js`

- 可解析`Data_netWorthTrend`完整净值序列。
- 免费，但属于公开网页接口，不承诺长期稳定。
- 只作AKShare失败时的后备。

官方文档：
https://akshare.akfamily.xyz/data/fund/fund_public.html

## 14:30盘中估值

1. AKShare `fund_value_estimation_em`
   - 返回当日估算值、估算增长率和上一公布净值。
   - 免费，底层数据来自东方财富。
2. 东方财富`fundgz.1234567.com.cn/js/{基金代码}.js`
   - 免费JSONP估值，非正式稳定API。
3. AKShare `fund_etf_spot_em`
   - 对ETF联接基金，用跟踪ETF的实时价格、IOPV、折溢价和资金流做独立确认。
4. 养基宝
   - 暂未发现稳定、公开、无需登录的官方API。
   - 用户导出的截图或数据可作为辅助，不能绕过登录抓取。

估值只在绝对涨跌不低于1.5%时进入判断。未完成至少20次历史校准前，
除非绝对涨跌不低于3%且ETF与成分股广度双确认，否则不触发申赎。

## 付费API结论

在每月30元以内，没有必要购买数据服务：

- 完整净值和ETF盘中数据可由AKShare免费获得。
- 本地缓存、重试、完整性校验和双源回退比低价单源订阅更可靠。
- Wind等机构终端适合一致性要求更高的专业场景，但不符合本项目预算。

## 数据纪律

- 每次下载保留日期、来源和完整性校验结果。
- 历史回测只使用当时已经披露的数据。
- 季报规模和持有人数据默认在报告期后90天才可用于模型。
- 盘中估值在官方净值发布后逐日校准，不正确的估值源自动提高触发阈值。
